# `prisma seed`

Seed the service with initial data.

#### Usage

```sh
prisma seed [flags]
```

#### Flags

```
-r, --reset       Reset the service before seeding
```

#### Examples

##### Seed service with initial data

```sh
prisma seed
```

##### Seed service with initial data, resetting the service first

```sh
prisma seed --reset
```
